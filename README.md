# LimonGram-messenger

Ваш защищённый мессенджер с Е2Е шифрованием. 

Предыстория создания: 
После некоторых событий, возникла острая необходимость в создании собственного мессенджера для комфортного общения. Работа над проектом продолжалась больше месяца и продолжается до сих пор. 

Для нас принципиально важно, чтобы пользователи получили максимально безопасный мессенджер для обычного общения, не для терроризма.

Мы будем улучшать сервис и создавать новые функции.
